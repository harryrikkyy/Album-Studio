🛠️
Utilities
One-click batch helpers and the Photoshop floating dock.

Pick a folder of high-res images. They get downscaled and saved into a new _Thumbnails subfolder. Use the proxies in the Red Box for lightning-fast UI performance.

Pick a folder of PSDs. Every file gets exported twice — once at quality 12 into JPEG-High-Res/ and once at quality 1 into JPEG-Low-Res/, both created as siblings of the source folder.

Editing dock Closed

The floating Tools Bar attaches to Photoshop, follows its window, and hides when Photoshop minimizes. Drop quick-access tools into it.

🏷️
Renamer
Rename an order folder to the print naming convention.

A drag-and-drop workspace for an order folder. Pick the first (00), last (zz), and cover sheets by clicking; the rest auto-number as <folder> (01). Drag to reorder, right-click a page for a lamination effect, then Rename All writes to disk collision-safe. Supports JPEG, PNG, and PSD/PSB.
